# Code Quality Analyzer

## Overview

A Python tool that analyzes source code quality metrics such as total lines, blank lines, and comment lines. This project demonstrates file handling and basic static code analysis techniques.

## Features

- Analyze Python source files
- Count total lines of code
- Count blank lines
- Count comment lines
- Generate simple quality reports

## Technologies Used

- Python

## Installation

```bash
python code_quality_analyzer.py
